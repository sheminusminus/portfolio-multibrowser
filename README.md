# portfolio-multibrowser
My portfolio site, with special behavior in VR browsers.

## run in development

```
git clone https://github.com/sheminusminus/portfolio-multibrowser
cd portfolio-multibrowser
npm i
npm run dev
```

## build for production

```
git clone https://github.com/sheminusminus/portfolio-multibrowser
cd portfolio-multibrowser
npm i
npm run build
```
